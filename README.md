# Tugas Jobsheet 3 - Pemrograman Mobile
Penjelasan: 
Tugas ini adalah berisi permainan sederhana bernama Game Tebak Angka. Program akan menentukan sebuah angka rahasia dan pengguna diminta untuk menebak angka tersebut melalui terminal.

Fitur Utama:
Perulangan (while loop): Program akan terus berjalan dan meminta input selama tebakan pengguna belum benar.
(if-else): Memberikan pesan kepada pengguna apakah tebakan terlalu tinggi, terlalu rendah, atau tepat.
Validasi Input: Memastikan angka yang dimasukkan adalah angka positif (lebih dari 0).
Penghitung Percobaan: Menampilkan total berapa kali pengguna mencoba menebak hingga berhasil.
